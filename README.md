# Road-Map Builder

Веб-приложение для создания и управления roadmap-проектами.

## Возможности

- **5 режимов отображения:**
  - Древовидный вид (Tree)
  - Временная шкала (Timeline)
  - Канбан доска (Kanban)
  - Диаграмма Ганта (Gantt)
  - Календарь (Calendar)

- **Управление задачами:**
  - Название, описание, статус, приоритет, даты
  - Теги для группировки
  - Подзадачи (subtasks)
  - Прогресс выполнения (0-100%)
  - Иерархия задач (родитель-потомок)

- **Инструменты:**
  - Drag-and-drop перемещение узлов
  - Поиск и фильтрация
  - Zoom/Pan (колёсик + Ctrl, Shift+drag)
  - Undo/Redo (Ctrl+Z / Ctrl+Y)
  - Экспорт в JSON и PNG
  - Импорт из JSON
  - Горячие клавиши

## Использование

1. Откройте `index.html` в браузере
2. Добавляйте задачи кнопкой "Добавить"
3. Переключайтесь между видами в тулбаре
4. Используйте правый клик на узел для быстрых действий

### Горячие клавиши

| Клавиша | Действие |
|--------|----------|
| `Del` | Удалить выбранную задачу |
| `Esc` | Закрыть модал |
| `Ctrl+N` | Новая задача |
| `Ctrl+S` | Экспорт JSON |
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` | Вернуть |
| `Ctrl+колёсико` | Zoom |
| `Shift+drag` | Pan |

## Технологии

- Чистый JavaScript (ES6+)
- HTML5 + CSS3
- LocalStorage для хранения
