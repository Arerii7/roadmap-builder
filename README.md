# 🗺️ Road-Map Builder

Веб-приложение для визуализации roadmap-проектов.

## Возможности

| Режимы | Управление | Инструменты |
|--------|------------|-------------|
| Tree • Timeline • Kanban • Gantt • Calendar | Задачи, подзадачи, теги, прогресс | Drag-n-drop, фильтры, undo/redo |

- **5 режимов отображения**
- **Иерархия задач** с parent-child
- **Теги** и **прогресс** (0-100%)
- **Zoom/Pan** — `Ctrl+колёсико`, `Shift+drag`
- **Горячие клавиши**: `Del`, `Esc`, `Ctrl+Z/Y`, `Ctrl+N/S`
- **Экспорт** JSON / PNG

## Быстрый старт

```bash
# Клонировать
git clone https://github.com/Arerii7/roadmap-builder.git

# Открыть в браузере
cd roadmap-builder && open index.html
```

## Использование

1. Открой `index.html` в браузере
2. Добавляй задачи кнопкой «Добавить»
3. Правый клик на узел — быстрые действия
